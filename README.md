👉 

##  Tech Stack
* React.js
* Tailwind CSS
* JavaScript (ES6+)
* Framer Motion
* Vite
  
## Features
* Responsive design (mobile + desktop)
* Smooth animations and transitions
* Project showcase with live demo & code links
* Contact section
* Resume download option

## 📂 Projects Included
* 🎥 YouTube-like Video Streaming Backend
* ✅ Todo Application (Context API + Local Storage)
* 💱 Currency Converter (API Integration)
* 🎨 PW Skills UI Clone
* 🌐 Portfolio Website

## ⚙️ Installation & Setup

### 1. Clone the repository
```
git clone https://github.com/rahul18kumar/portfolio.git
```
### 2. Navigate to project
```
cd portfolio
```
### 3. Install dependencies

```
npm install
```

### 4. Run locally
```
npm run dev
